O que é HTML?

HTML significa "HyperText Markup Language" (Linguagem de Marcação de Hipertexto), é a linguagem usada para criar páginas na internet.

Pense no HTML como a receita de um bolo. Quando você escreve uma receita, você inclui títulos, listas de ingredientes e instruções. No HTML, você faz a mesma coisa, mas com "tags" que dizem ao navegador como organizar e mostrar o conteúdo na tela.

Apronfundamento de Tags

Lista Não Ordenada (ul) Lista Ordenada (ol) Item da Lista (li)
Pense nisso como uma lista de itens onde a ordem não importa. É como se você estivesse anotando itens aleatórios. EX:Uma lista de compras onde a ordem dos itens não importa. Esta é uma lista onde a ordem dos itens importa. É como se você estivesse numerando os passos de uma receita ou uma sequência de tarefas. EX:Uma receita de bolo onde você precisa seguir os passos em uma ordem específica. Esta é a "etiqueta" que você usa para cada item individual em ambas as listas acima (não ordenada e ordenada). É como se cada linha da sua lista fosse um item separado. Ex:Cada ingrediente da sua lista de compras ou cada passo da receita

Exemplos

Lista não ordenada

  • Calça
  • Meia
  • Moletom

Lista ordenada

  1. 2 Ovos
  2. 1 xícara Leite
  3. 1 xícara Óleo
