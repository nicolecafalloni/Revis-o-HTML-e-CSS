O que é CSS?

CSS significa Cascading Style Sheets, é uma ferramenta usada para definir o visual e o layout de páginas da web.

O CSS é como a "roupa" e o "maquiagem" que a página usa, determinando como ela vai parecer para os visitantes, em resumo, o CSS ajuda a deixar as páginas da web bonitas e organizadas.

O que o CSS pode fazer?

  • Mudar a cor do texto ou do fundo.
  • Alterar o tamanho e o tipo das fontes.
  • Ajustar o espaçamento entre elementos.
